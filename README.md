# Editor de Texto con Pilas (Undo / Redo)

## Objetivo
Implementar una estructura de pila en Java para simular un sistema de deshacer y rehacer en un editor de texto.

## Funcionalidades

- Escribir texto
- Deshacer acciones
- Rehacer acciones
- Mostrar texto actual

## Tecnologías
Java
IntelliJ IDEA
GitHub

## Instrucciones de ejecución

1. Compilar los archivos Java
2. Ejecutar Main.java
3. Usar el menú en consola
